# JetBrains Theme

A dark theme for VS Code inspired by JetBrains IDEs (IntelliJ IDEA, WebStorm, PyCharm, etc.).

## Installation

1. Open VS Code extensions (`Ctrl+Shift+X`).
2. Search for **JetBrains Theme**.
3. Install and select **JetBrains Dark Theme**.

## Supported Languages

Optimized highlighting for:
- JavaScript, TypeScript, React, HTML, CSS
- Java, Python, Go, Rust, Bash
- JSON, YAML, TOML, XML, Jenkinsfile

## Colors

| Scope | Color | Hex |
|-------|-------|-----|
| Comments | Gray | `#7a7e85` |
| Text | Light Gray | `#bcbec4` |
| Numbers | Cyan | `#2aacb8` |
| Keywords | Orange | `#cf8e6d` |
| Strings | Green | `#6aab73` |
| Functions | Blue | `#55a6f2` |
